# BSE 500 Exclusion Note

## Decision

BSE 500 is excluded from the current-snapshot empirical dataset for now.

## Reason

The current repository includes only index snapshots for which constituent or top-weight data can be traced to a clean official source.

For BSE Sensex, an official BSE scripwise-weightage source was available and was therefore included as a full-constituent snapshot.

For BSE 500, a comparably clean official current scripwise-weightage table has not yet been verified. Therefore, BSE 500 is not included in the current empirical tables to avoid contaminating the dataset with uncertain or non-comparable data.

## Paper Treatment

The paper should state:

> BSE 500 is identified as a potential extension but excluded from the current empirical sample because a verified public scripwise-weightage source comparable to the Sensex dataset could not be confirmed at this stage.

## Future Extension

If a verified official BSE 500 scripwise-weightage source becomes available, it can be added under the same data structure used for Sensex.
