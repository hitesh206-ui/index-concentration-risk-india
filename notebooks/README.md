# Notebooks

Planned notebook workflow:

1. `01_data_collection_and_cleaning.ipynb` - assemble index factsheet data and clean constituent/sector weights.
2. `02_concentration_metrics.ipynb` - calculate HHI, top-N weights, effective N, sector HHI, and Theil entropy.
3. `03_risk_linkage_analysis.ipynb` - connect concentration metrics with index volatility and drawdowns.
4. `04_global_comparison.ipynb` - compare Indian indices with global benchmarks.
5. `05_sebi_framework_analysis.ipynb` - evaluate concentration-risk gaps under the SEBI Significant Indices framework.
6. `06_charts.ipynb` - generate publication-ready figures.

Notebook files will be added after the first data template and pilot dataset are prepared.
